```
分布式系统的复杂性增加也将导致更高的网络故障率。
优雅的服务降级
变更管理
健康检查与负载均衡
自我修复
故障转移缓存
重试逻辑
限流器和负载降级
快速失败原则与独立性
舱壁模式
断路器
测试故障
```

```

动态环境和分布式系统（如微服务）会导致更高的故障机率；
服务应该做到故障隔离，到达优雅降级，来提升用户体验；
70％的中断是由变化引起的，代码回滚不是一件坏事；
做到服务快速失败与独立性。团队是无法控制他们所依赖的服务情况；
缓存、舱壁、断路器和限流器等架构模式与技术有助于构建可靠的微服务架构。

```
