title:  vi之禅
---

Satish Reddy

```
To me vi is Zen. 
To use vi is to practice Zen. 
Every command is a koan. 
Profound to the user, unintelligible to the uninitiated. 
You discover truth every time you use it.”

```


```
vi 之大道如我心之禅，
vi 之漫路即为修禅，
vi 之命令禅印于心，
未得此道者视之怪诞，
与之为伴者洞其真谛，
长修此道者巨变人生。

```
